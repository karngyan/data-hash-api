# Data Hash Api

Hosted version: 

## Local development

```sh
yarn
yarn dev
```

The api will be served from `http://localhost:3000/dataWithHash`

## Deployment

This api is set up to be deployed using Zeit Now.

Install the cli:

```sh
npm i -g now
```

Deploy:

```sh
now
```

You will have to create an account and/or log in first.
